Sobre o Site

Este é um projecto desenvolvido para fins académicos que visa oferecer uma plataforma para análise de risco de eventos desportivos.

Funcionalidades

  • Entrar - Acesso à conta pessoal.
  • Registar - Criar uma conta.
  • Análise de Risco - Uma simulação de como poderia ser o output de uma análise de risco.
  • Registar Evento - Adicionar novos eventos desportivos.

Contacto

Para mais informações ou assistência, por favor, entre em contacto connosco através do email [email@example.com].

Marcas HTML usadas:

  • Marca <!DOCTYPE html> para declarar o tipo de documento como HTML5.
  • Marca <html> que é a raiz do documento HTML.
  • Marca <head> que contém metadados como <meta>, <title> e <link>.
  • Marca <meta charset="UTF-8"> para definir a codificação de caracteres para o documento.
  • Marca <meta name="viewport"> para controlar a área de visualização da página.
  • Marca <title> para definir o título da página.
  • Marca <link> para ligar folhas de estilo externas, definir e definir o favicon.
  • Marca <body> que contém todo o conteúdo da página como <header>, <main> e <footer>.
  • Marca <header> para a cabeça do documento.
  • Marca <div> usada no cabeçalho para conter o menu (id="menu-container").
  • Marca <section> usada para organizar conteúdo por temas.
  • Marca <a> usada para criar links clicáveis.
  • Marca <p> para parágrafos de texto.
  • Marca <ul> para listas não ordenadas.
  • Marca <li> para itens de lista.
  • Marca <script> para incluir scripts JavaScript, como o do menu.
  • Marca <footer> para o rodapé do documento.

Propriedades CSS usados:

  • padding: Aplicado a main e section para adicionar espaço dentro dos elementos, melhorando a legibilidade e o conforto visual.
  • max-width: Limita a largura máxima do elemento main a 800px para garantir que o conteúdo seja fácil de ler em dispositivos largos.
  • margin: Centraliza o elemento main horizontalmente na página e adiciona margem abaixo de cada section para separar visualmente as seções.
  • min-height: Define a altura mínima do elemento main como a altura da viewport menos 230px, assegurando que main ocupa uma quantidade adequada de espaço na tela. Estavamos com dificuldade em igualar a localização do footer em todos os documentos, pelo que também auxiliou neste expediente
  • background: Define a cor de fundo das section como branca, destacando-as contra outras cores de fundo da página.
  • border-radius: Arredonda os cantos das section com um raio de 8px para um design mais suave e moderno.
  • box-shadow: Adiciona sombras subtis abaixo das section para criar uma sensação de elevação e componente estética.
  • text-decoration: Remove o sublinhado dos links (a) para um visual mais limpo, e adiciona sublinhado ao passar o mouse (a:hover) para indicar interatividade.
  • list-style-type: Remove os marcadores padrão de listas (ul) para um design mais limpo.
  • margin-bottom: Adiciona uma margem na parte inferior de cada section para criar espaço entre as seções consecutivas, em vista melhorar a separação visual e a organização do conteúdo.
  • font-family: Determina a família de fontes dos elementos, proporcionando uma aparência consistente e suporte a diferentes sistemas operacionais e dispositivos.
  • text-align: Alinha o texto dentro de um elemento, usado aqui para justificar ou centralizar o texto, dependendo do contexto.
  • background: Define a cor de fundo dos elementos, usada para criar contraste visual e destacar áreas específicas da página.
  • color: Especifica a cor do texto, garantindo legibilidade e contraste adequados com o fundo.
  • width: Controla a largura de um elemento, aqui usado para estender o rodapé por toda a largura da viewport.
  • border-radius: Arredonda os cantos de elementos, como imagens e seções, para um visual mais suave e moderno.
